=============================================================================
TROUBLESHOOTING
=============================================================================

If you get "large file" errors:
--------------------------------
# Check file sizes
du -h models/pppq/*.txt

# Install Git LFS if needed
git lfs install
git lfs track "models/pppq/*.txt"
git lfs track "models/pppq/*.json"
git add .gitattributes
git commit -m "chore: Add Git LFS for model files"
git push origin main

# Then retry the main push


If you want to exclude model files:
------------------------------------
# Add to .gitignore
echo "models/pppq/*.txt" >> .gitignore
echo "models/pppq/*.json" >> .gitignore
git add .gitignore
git commit -m "chore: Ignore large model files"

# Note: This means models won't be in git, which is usually fine
# You can retrain them with: python src/models/pppq_multi_output_model.py
